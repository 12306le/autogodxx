### 获取mac地址()

> 获取 MAC 地址
> 首先尝试从 WifiManager 获取，如果获取到的是伪造的 MAC 地址则视为无效
> 若无效则尝试通过网络接口获取，若还是无效且系统版本大于等于 Android 7.0（N），
> 则尝试从文件中读取

- 返回 : MAC 地址，如果无法获取则返回 null
- 版本 : 1.7.0


```javascript
// 获取mac地址
日志("mac地址:",$设备.获取mac地址());
```
