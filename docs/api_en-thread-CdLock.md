# CdLock - 倒计时锁

- 更新时间:2026-01-30 19:32:56

> 倒计时锁
> 
> 是对CountDownLatch的封装对象，倒计时锁，用于等待多个线程完成。




### ok()

> 完成一个倒计时

- 版本 : 1.8.3


### finish()

> 等待所有倒计时完成

- 版本 : 1.8.3


### finish(timeout)

> 等待所有倒计时完成

- 参数 : timeout {long} 超时时间(ms)
- 返回 : {boolean} 是否完成
- 版本 : 1.8.3


### getCount()

> 获取倒计时数量

- 返回 : {long} 倒计时数量
- 版本 : 1.8.3
