# AppTaskInfo - 任务信息

- 更新时间:2026-01-30 19:32:51

> 应用任务信息
> 
> 通过 $app.lsRecent(); 方法返回的列表元素类型




### const {String} 应用名称;

> 应用名称


### const {String} 包名;

> 包名


### const {String} 最后使用时间;

> 最后使用时间


### const {long} 最后使用的时间;

> 最后使用时间


### const {ApplicationInfo} 原生信息;

> 安卓原生应用信息
> 
> 类型：{android.content.pm.ApplicationInfo}


### 获取时间戳()

> 最后使用时间

- 返回 : {long} 最后使用时间
- 版本 : 1.7.0


### 获取应用名()

> 获取应用名称

- 返回 : {字符串} 应用名称
- 版本 : 1.7.0


### 获取包名()

> 获得包名

- 返回 : {字符串} 包名
- 版本 : 1.7.0


### 获取时间()

> 获得最后时间格式化字符串

- 返回 : {字符串} 最后使用时间
- 版本 : 1.7.0


### 获取原生对象()

> 获取安卓原生应用信息

- 返回 : {android.content.pm.ApplicationInfo} 安卓原生应用信息
- 版本 : 1.7.0


### json()

> 将当信息对象转为json字符串

- 返回 : {String} json字符串
- 版本 : 1.7.0
