# $hid - 键鼠触控

- 更新时间:2026-01-30 19:32:53

> 键鼠触控
> 
> 无需任何权限，连接键鼠硬件即可完成自动化触控功能。




### 初始化()

> 初始化HID

- 返回 : {字符串} 初始化成功返回true，否则就是失败原因
- 版本 : 1.8.9


```javascript
定义 结果 = $键鼠.初始化();
如果(结果=="true"){
    日志("初始化成功");
}否则{
    日志("初始化失败:",结果);
}
```
