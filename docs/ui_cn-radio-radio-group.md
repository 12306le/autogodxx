# 单选按钮组

- 更新时间:2026-01-30 19:33:13

> 单选按钮组-radio-group
> 
> 在此布局中，单选按钮会自动取消和选中，保证一组按钮只被选中一个
> 
> 是布局父类的(XLayout)的子类
> 
> 原生类型:{android.widget.RadioGroup}




## 一、常用属性


### 选中(下标)

> 设置选中的下标

- 参数 : 下标 {int} 按钮下标


```javascript
//解析布局,获得界面对象
定义 界面 = $界面.解析布局("./资源/主界面.xml");
//获得控件
定义 单选组 = 界面.找控件("我的单选组");
//设置选中的下标
单选组.选中(0);
```


### 监听选中(回调函数)

> 监听选中按钮的下标

- 参数 : 回调函数 ((下标)=>{}) 回调函数


```javascript
//解析布局,获得界面对象
定义 界面 = $界面.解析布局("./资源/主界面.xml");
//获得控件
定义 单选组 = 界面.找控件("我的单选组");
//监听选中按钮的下标
单选组.监听选中((下标)=>{
    $界面.土司("当前下标:"+下标);
});
```


### 获取选中下标()

> 获取选中的下标

- 返回 : {int} 下标 (-1表示没有选择)


```javascript
//解析布局,获得界面对象
定义 界面 = $界面.解析布局("./资源/主界面.xml");
//获得控件
定义 单选组 = 界面.找控件("我的单选组");
//获得选中的下标
定义 下标 = 单选组.获取选中下标();
```

//解析布局,获得界面对象
