## 二、常用函数


### 设置粗细(粗细)

> 设置粗细

- 参数 : 粗细 {int} 粗细(px)


```javascript
//解析布局,获得界面对象
定义 界面 = $界面.解析布局("./主界面.xml");
//获取控件
定义 分割线 = 界面.找控件("我的分割线");
//设置粗细
分割线.设置粗细(20);
```
