### 2. 名称颜色

| 属性值              | 颜色        |
|------------------|-----------|
| white、白色         | #FFFFFF   |
| black、黑色         | #000000   |
| null、透明、none、空、无 | #00000000 |
| green、绿色         | #57965C   |
| red、红色           | #C94F4F   |
| blue、蓝色          | #3574F0   |
| yellow、黄色        | #C29E4A   |

:::tip 注意
由于脚本支持中文和英文编程，所以颜色属性值可以采用中文，也可以采用英文。

例如：color="red"

等同：颜色="红色"
:::
