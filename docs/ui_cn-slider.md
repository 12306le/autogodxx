## 二、常用函数


### 设置背景颜色(颜色)

> 设置背景颜色

- 参数 : 颜色 {数字} 文本颜色值
- 版本 : 1.8.2


```javascript
//解析布局,获得界面对象
定义 界面 = $界面.解析布局("./资源/主界面.xml");
//获取控件
定义 滚动条 = 界面.找控件("我的滚动条");
//设置背景颜色
滚动条.设置背景颜色($颜色.红色);
```

### 监听变化(回调)

>

- 参数 : 回调 {(数值)=>{}}


```javascript
//解析布局,获得界面对象
定义 界面 = $界面.解析布局("./资源/主界面.xml");

    //..
});
```


### 获取值()
