# FcConfig - 文件选择配置

- 更新时间:2026-01-30 19:32:52

> 文件选择配置




### const {String} mode;

> 选择模式:多选或者单选(single,multi)


### const {String} type;

> 选择类型:文件还是目录,还是两者都(file,mkdir,all)


### const {String} path;

> 默认路径 /sdcard


### const {String} root;

> 最高根路径 /sdcard


### const {String} ext;

> 后缀过滤(使用逗号分割)
