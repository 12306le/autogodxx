- 更新时间:2026-01-30 19:33:13

> 
> 原生类型:{com.google.android.material.floatingactionbutton.FloatingActionButton}

## 二、常用函数


### 显示()

> 显示悬浮按钮


```javascript
//解析布局,获得界面对象
定义 界面 = $界面.解析布局("./主界面.xml");
//获取控件
定义 悬浮按钮 = 界面.找控件("我的悬浮按钮");
//显示控件
悬浮按钮.显示();
```
